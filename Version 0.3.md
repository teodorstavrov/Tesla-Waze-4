# TesRadar — Version 0.3

**Първото уеб навигационно приложение в България, специализирано и оптимизирано за браузърите на Tesla автомобили.**

Безплатно. Без реклами. Без регистрация.

---

## Накратко

TesRadar е навигационен асистент в реално време за шофьори на Tesla в България. Показва EV зарядни станции от три источника, следи 47 отсечки за средна скорост по магистрали и пътища от I/II клас, предупреждава за пътни инциденти докладвани от общността (полиция, катастрофи, опасности), и изгражда маршрути с изчисление на батерията при пристигане. Работи изцяло в браузъра — без инсталация, без акаунт.

---

## Карта и навигация

### Интерактивна карта
- Базирана на OpenStreetMap чрез Leaflet
- Три режима на картата: **Тъмна** / **Светла** / **Сателит**
- Автоматично превключване тъмна/светла тема по часа на деня (проверява на 60 мин)
- Ръчно превключване с бутон горе вляво

### GPS и ориентация
- GPS проследяване в реално време (Web Geolocation API)
- Цветово-кодирана точност: зелено < 20 м · жълто < 50 м · оранжево < 100 м · червено ≥ 100 м
- Два режима:
  - **Север нагоре** — картата е статична, GPS аватарът се върти по посоката
  - **Курс нагоре** — аватарът винаги сочи нагоре, картата се върти плавно (0.25 s transition)
- GPS пулсиращ ореол (анимация 2.2 s) около аватара
- Бутон за центриране — връща картата към текущата позиция

### Маршрутизиране
- Маршрут от текущата позиция до търсена дестинация (OSRM)
- Визуален HUD с текущата инструкция за завой и разстояние
- Панел с: оставащи км · времетраене · ETA (час на пристигане)
- До 3 алтернативни маршрута — таб за превключване с разстояние
- Автоматично преизчисляване при отклонение > 200 м (ръчен бутон „Пренасочи")
- EV станции покрай маршрута с разстояние от трасето — кликни за навигация
- Прогноза на батерията: текущ % → очакван % при пристигане (зелено/жълто/червено)

## Търсене на адреси

- Поле за търсене (иконка горе в центъра) — разширява се при докосване
- 400 мс debounce — не спамва Nominatim API
- Паралелно търсене: геокодиране (Nominatim/OSM) + локални EV станции
- Резултати в две секции: „Зарядни станции" и „Места"
- **История на търсенето** — последните 10 търсени дестинации (localStorage)
- **Любими места** — звезда ⭐ до всеки резултат; добавя/премахва от любими
  - Любимите се показват над историята при отваряне на търсачката
  - ✕ бутон за изтриване от историята (само не-любими записи)
- Резултатите са кеширани 10 мин на клиента — без повторни заявки

## EV Зарядни станции

### Данни и источници
- Три интегрирани источника:
  - **Tesla** (червен маркер) — Tesla Supercharger мрежа
  - **OCM** (син маркер) — OpenChargeMap
  - **OSM** (зелен маркер) — OpenStreetMap
- Данните се синхронизират седмично от cron job и се кешират в Redis
- Клиентски localStorage кеш (24 ч) — станциите се зареждат веднага дори без интернет

### Маркери на картата
- Голям „⚡" divIcon (40×40 px) — лесно докосване на Tesla екран
- Цвят на фона по статус: зелено (свободна) · оранжево (заета) · червено (офлайн) · сиво (неизвестно/планирана)
- Само в рамките на видимата карта (bbox filtering)

### Панел с детайли (при докосване)
- Име, адрес, град
- Badges: источник (TESLA/OCM/OSM) и статус (Свободна/Заета/Офлайн/Планирана)
- Мрежа оператор, брой порти (свободни/общо), максимална мощност (kW), цена (€/kWh или Безплатно)
- Списък конектори: тип + мощност + брой (напр. CCS 150 kW ×2)
- Дата на последно обновяване
- **Бутон „Обнови"** — ръчен refresh, заобикаля server кеша, чете директно от Redis (`?bust=1`)
- Бутон „↗ Навигирай" — задава станцията като дестинация

### Филтриране
- Лента с филтри (появява се под търсачката когато маркерите са видими)
- Конектор (едновременно само един): Tesla · CCS · CHAdeMO · Type 2
- Минимална мощност: 50 kW+ · 150 kW+
- Наличност: само свободни
- Бутон „✕ Изчисти" при активни филтри

## Камери за средна скорост

### База данни — 47 отсечки
Покрити пътища:
- **А1 Тракия** — 14 секции (двете посоки, включително Пловдив bypass chain)
- **АМ Хемус (А2)** — 3 секции
- **АМ Струма (А3)** — 6 секции (двете посоки)
- **А4 Марица** — 6 секции (двете посоки)
- **АМ Европа (А6)** — 1 секция (120 км/ч)
- **Пътища І-1, І-2, І-3, І-4, І-5, І-6, ІІ-55** — 17 секции (90 км/ч)

Лимит: 140 км/ч на магистрали · 90 км/ч на I клас · 80 км/ч камиони

### Карта
- Оранжева пунктирана линия за всяка отсечка
- Активна отсечка — червена плътна линия
- 📷 маркери на начало и край с лимита
- Tooltip (не-Tesla): „А1 Тракия · Вакарел — Ихтиман — 140 км/ч (19 км)"

### Предупреждение при приближаване (2 км)
- Карта (горе вдясно) показва „⚠️ Зона за средна скорост" с:
  - Наименование на пътя и участъка
  - Разстояние до начало (напр. „1.8 км напред")
  - Speed limit badge (червен кръг)
- Тихо бипване (550 Hz) при влизане в 2 км зона
- Автоматично изчезва при влизане в активната отсечка

### Активна отсечка (SectionCard)
- **Голям цифров дисплей на средната скорост** (38 px, цветово кодирана):
  - Зелено — под лимита
  - Оранжево — над 93% от лимита
  - Червено — над лимита
- **Progress bar** — показва изминатото разстояние в отсечката (синьо/оранжево/червено)
- **Оставащи км** вдясно под progress bar-а
- Speed limit badge (червен кръг) + изминало време
- Предупреждение „⚠️ Намали скоростта!" при превишаване
- Бипване на 30 сек интервали при превишена скорост (само не-Tesla)

### Резултат след края
- ✅ „В рамките на лимита" (зелено) или 🚨 с окончателната средна скорост (червено)
- Показва се 15 секунди, после изчезва

### Алгоритъм
- Средна скорост = `дължина на отсечката (м) / изминало време (сек) × 3.6`
- Правна мярка — пътна дължина, не GPS-акумулирано разстояние
- GPS стъпки > 200 м се игнорират (телепортация/грешка)
- Предварителен bbox filter (10 км) преди haversine изчисление

## Пътни събития (общностен репортинг)

### Типове маркери
| Тип | Цвят | Иконка | Алертна зона |
|-----|------|--------|--------------|
| Полиция | Синьо | 🚔 | 700 м |
| Катастрофа | Червено | 🚨 | 600 м |
| Опасност | Оранжево | ⚠️ | 500 м |
| Задръстване | Лилаво | 🚗 | 800 м |
| Камера | Оранжево | 📷 | 600 м |
| Строителство | Оранжево | 🚧 | 500 м |

### Как работи репортването
- Докоснете червения бутон „Докладвай" в центъра на дока
- Изберете тип (4 налични от UI: полиция, катастрофа, опасност, камера)
- Локация: GPS позиция → ако няма GPS, центъра на картата
- Геофенс валидация: само в рамките на България (41–44.5°N, 22–28.7°E)
- Rate limit: 20 доклада на IP на 10 минути
- Маркерите са постоянни — изтриват се само при гласуване или от администратор

### Гласуване (потвърди / отхвърли)
- EventPanel се отваря автоматично при влизане в 80 м зона около маркер
- Бутон „👍 Потвърждавам" — затвърждава присъствието
- Бутон „👎 Вече го няма" — един глас → маркерът се изтрива автоматично
- Панелът се затваря автоматично след 20 секунди
- Cooldown 3 мин за повторно отваряне на същия маркер

### Аудио предупреждения
**Всички типове — zone-entry detection (само при приближаване):**
- Предупреждението се пуска точно веднъж при влизане в зоната
- Автоматично нулиране при излизане → при следващо приближаване предупреждава отново
- Без spam при отдалечаване

**Полиция — два прага:**
1. **700 м** — бипване (880 Hz, 150 мс) + toast + глас „Полиция напред" (не-Tesla)
2. **300 м** — сирена (3 сек) + toast „Полиция — 300 метра!" + глас (не-Tesla)

**Останали типове — 1 праг:**
- Бипване (880 Hz, 100 мс) + toast с текст + глас на не-Tesla устройства

### Toast известия
- Появяват се горе вдясно (opacity + scale анимация)
- Показват: иконка на типа, текст, разстояние в метри
- Автоматично изчезват след 12 секунди
- Бутон „✕" за ръчно затваряне

## Аудио система

- **Web Audio API** — работи директно в браузъра на Tesla без плъгини
- Автоматично отключване при първо докосване на екрана (browser autoplay policy)
- Бутон за заглушаване (горе вдясно в stats карта)
- Функции:
  - `beep(frequency, duration)` — кратко бипване за камери и маркери
  - `siren(count)` — пулсираща сирена за полиция
  - `speak(text)` — синтез на реч на български (само не-Tesla)
- Tesla браузър: само beep и siren — речта е деактивирана (твърде бавна в QtWebEngine)

## Интерфейс и UX

### Bottom Dock (долна лента)
Три основни бутона винаги видими:
- **⚡ Станции** (ляво) — показва/скрива EV маркерите (0.35 opacity при скрити)
- **🔴 Докладвай** (център, червен) — отваря модала за репортване
- **↗ Маршрут** (дясно) — показва RoutePanel или съобщение „Няма маркиран маршрут"

### Плаващи контроли
- **Горе ляво:** Тема · Сателит · Ориентация · GPS центриране
- **Горе дясно:** Stats карта (звук · брой станции · брой събития · GPS точност)
- **Горе в центъра:** Търсачка
- **Горе вдясно (при активна отсечка):** SectionCard с дисплей на средна скорост

### Glassmorphism дизайн
- Полупрозрачни панели с blur ефект (`rgba(22,22,30,0.92)` + `backdrop-filter: blur`)
- Тъмна/светла тема с CSS variables
- На Tesla: backdrop-filter е деактивиран (принуждава цял-екран repaint)

### Анимации
- Попъп прозорци: opacity + scale(0.96→1) — само GPU операции, без layout
- Progress bar на отсечката: `width` transition 1 s linear (деактивирано на Tesla)
- Toast: opacity + scale появяване
- GPS пулс: безкрайна анимация

### Tesla оптимизации
- Всички постоянни панели (EventPanel, StationPanel, SectionCard) са винаги в DOM
- Видимостта се контролира с `aria-hidden` — без mount/unmount jitter
- CSS transitions = 0 s на Tesla (без трептене)
- GPU layer pre-allocation при зареждане
- `contain: layout style paint` на диалоги — предотвратява map repaints

## Backend архитектура

### API endpoints
| Endpoint | Метод | Описание |
|----------|-------|---------|
| `/api/ev/stations` | GET | Всички EV станции за bbox; `?bust=1` заобикаля кеша |
| `/api/events` | GET | Всички активни събития в България (90 сек кеш) |
| `/api/events` | POST | Нов доклад (rate limit 20/10 мин) |
| `/api/events/confirm` | POST | Потвърди събитие |
| `/api/events/deny` | POST | Отхвърли събитие (1 глас → изтриване) |
| `/api/cron/sync-stations` | GET | Седмичен sync от Tesla/OCM/OSM → Redis |
| `/api/admin/events` | GET/POST/DELETE | Admin CRUD без rate limit |

### Кеш стратегия
- **In-memory (Vercel instance):** events 90 сек · stations 20 мин
- **Redis:** events постоянно · stations седмичен snapshot
- **Client localStorage:** stations 24 ч · search history · favorites · vehicle profile
- **Client memory:** nominatim резултати 10 мин (до 100 записа)
- Единичен cache key `events:bg` за цяла България — без per-bbox фрагментация

### Rate limiting
- EV stations: 60 заявки/IP/мин (in-memory)
- Events GET: 60 заявки/IP/мин (in-memory)
- Events POST: 20 доклада/IP/10 мин (Redis)

### Технологии
- **Frontend:** React 19 · TypeScript · Vite · Leaflet
- **Backend:** Vercel Serverless Functions (Node.js)
- **Database:** Redis (Upstash)
- **Геокодиране:** Nominatim (OSM)
- **Маршрутизиране:** OSRM
- **Мониторинг:** Sentry
- **Хостинг:** Vercel (teslaradar.tech)
